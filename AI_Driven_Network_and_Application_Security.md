# AI-Driven Network and Application Security Reference List

This document provides a curated list of resources, tools, vendors, and educational materials focused on the intersection of Artificial Intelligence (AI) and Cybersecurity, specifically for network and application security.

## üåü Key Trends & Insights (2024-2025)

The landscape is evolving rapidly. Key themes include:

*   **Autonomous AI Agents:** The rise of autonomous agents for both offense (cyberattacks with minimal human intervention) and defense (automated SOCs).
*   **LLM Vulnerabilities:** Increasing focus on Prompt Injection, Sensitive Information Disclosure, and Supply Chain vulnerabilities (OWASP Top 10 for LLMs).
*   **AI-Orchestrated Attacks:** AI used to research targets, adapt ransomware, and execute sophisticated campaigns.
*   **Defensive AI:** Widespread adoption of AI for threat detection, predictive analytics, and automated response.

### Tools & Frameworks
*   **[Zeek](https://zeek.org/)**: Powerful network analysis framework that integrates with AI/ML for anomaly detection.
*   **[Wazuh](https://wazuh.com/)**: Open-source security platform with unified XDR and SIEM capabilities, increasingly leveraging AI.
*   **[Snort 3](https://www.snort.org/)**: The next generation of the Snort IPS, designed to be more extensible and performant, often used as a data source for AI models.
*   **[LlamaGuard](https://github.com/meta-llama/PurpleLlama)**: Safeguard models for LLM inputs and outputs.
*   **[Garak](https://github.com/leondz/garak)**: LLM vulnerability scanner.

## üè¢ Top AI Security Vendors

### Network & Endpoint Security
*   **Darktrace**: Pioneer in Self-Learning AI for network threat detection and autonomous response.
*   **CrowdStrike**: Falcon platform uses AI for real-time endpoint protection and threat hunting.
*   **SentinelOne**: Singularity platform offers autonomous AI-driven XDR.
*   **Vectra AI**: Specializes in AI-driven network detection and response (NDR).
*   **Palo Alto Networks**: Cortex XSIAM and XDR utilize AI for broad security operations.
*   **Fortinet**: FortiAI and FortiGuard Labs integrate AI across their security fabric.

### AI Application Security (Emerging)
*   **Protect AI**: Focuses on securing the AI lifecycle (MLSecOps).
*   **Lakera**: Solutions for protecting LLMs against prompt injection and other attacks.
*   **HiddenLayer**: Security for AI models (MLDR).
*   **Credo AI**: Governance and compliance for AI.
